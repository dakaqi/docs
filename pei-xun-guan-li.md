#### 发布培训

| 标题 | 无备注 |
| :--- | :--- |
| 内容 | 无备注 |
| 开始时间 | 无备注 |
| 结束时间 | 无备注 |
| 所需人数 | 大于0 |
| 地点 | 地址由常用地址管理功能提供，也可以通过快捷入口创建新的地址。 |
| 参与条件 | 选填。1. 本服务队会员 2. 本服务队准会员 3. 无限制 |
| 模式 | 1. 抢单模式 2. 竞标模式 |
| 酬劳/费用 | 选填。向参加者要求付费或者支付费用。 |

> 发送人：公益小助手
>
> 消息内容：您报名参加的培训\[培训活动名称\]取消了，取消原因：\[原因\]，给您带来不便请见谅。
