### 专区管理

#### 开设专区

输入内容：

| 专区名称 | 无备注 |
| :--- | :--- |
| 专区描述 | 30字以内 |

#### 查询专区

查询下设的所有专区以及分区信息。某一专区下，可创建分区。

显示内容：

| 专区名称 | 无备注 |
| :--- | :--- |
| 专区描述 | 无备注 |
| 下设分区名称 | 无备注 |
| 下设分区描述 | 无备注 |

#### 修改专区信息

修改专区的名称以及描述信息。

可修改内容：

| 专区名称 | 无备注 |
| :--- | :--- |
| 专区描述 |  |

#### 取消专区

取消某一个特定专区。取消需进行手机授权，确保非误操作造成的专区取消。
