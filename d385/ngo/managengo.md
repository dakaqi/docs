#### 组织结构管理

以年度方式展示某一个年度的所有职位列表，可通过选择年度的方式查看各个年度的组织结构。

列表显示内容：

| 职位 | 职位名称 |
| :--- | :--- |
| 姓名 | 负责该职位的会员姓名 |
| 会员编号 | 负责该职位的会员编号 |
| 状态 | 已审核生效、已提交待审核、历史 |
| 任期 | 该职位的任期 |
| 操作入口 | 待审核状态：1. 通过审核 2. 退回申请 |
|  | 已审核生效：修改任期 |

审批流程：

![](/assets/泳道流程图（垂直）.png)

服务队设定完成所有职位后，点击提交审核。

审核操作：

1. 通过审核  
   该组织结构生效，同时发送消息：

   > 发送人：组织小助手  
   > 内容：您被指派为某某某机构的某职务，操作人：某某某

2. 退回审核
